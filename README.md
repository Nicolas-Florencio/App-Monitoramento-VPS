# App-Monitoramento-VPS
Como projeto da matéria de Desenvolvimento de Dispositivos Móveis, criamos um app com pilar em monitoramento de link assim como Zabbix e PRTG. Será desenvolvido uma API em Node onde será possível adicionar novos links em monitoramento e por meio de pings e demonstrando se o serviço está disponível junto de sua latência e exibir um alerta
